## 📖 Overview

This repository contains the interactive frontend website for **Sri Pandian Kalyana Mahal**, a grand and vibrant event venue located in Madurai, Tamil Nadu. Designed to provide a luxurious first impression to potential clients, the website features stunning scroll animations, an auspicious color palette, responsive design, and seamless WhatsApp integration for booking inquiries.

## ✨ Key Features

- **Dynamic Animations:** Immersive scroll reveals, smooth parallax background effects, and an interactive sticky navbar.
- **WhatsApp Integration:** A custom inquiry form that directly forwards leads and booking requests to WhatsApp with pre-filled event details.
- **Interactive Location Map:** Deep integration with Google Maps to seamlessly guide guests to the venue in Iyer Bungalow, Madurai.
- **Premium UI/CSS:** An auspicious and vibrant design theme (`#c99339` Gold, `#8c1c13` Deep Red, `#f28500` Saffron) implemented with vanilla CSS avoiding heavy frameworks for rapid load times.
- **Fully Responsive:** Beautifully optimized across Desktop, Tablet, and Mobile screens.

## 🛠️ Technology Stack

- **Structure:** Semantic HTML5
- **Styling:** Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid)
- **Logic & Interactivity:** Vanilla JavaScript (`IntersectionObserver` API for scroll animations)
- **Icons & Typography:** Google Fonts (`Outfit`, `Playfair Display`) and inline SVG icons.

## 🚀 How to Run Locally

Since this is a lightweight frontend project without Node.js dependencies:

1. Clone this repository to your local machine.
2. Navigate to the `mahal-website` directory.
3. Simply open `index.html` in any modern web browser (Chrome, Edge, Firefox, Safari).
   *No build steps, bundled servers, or package installations are required!*
